# Sales Revenue Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes sales and revenue performance using interactive visualizations.

## KPIs
- Total Sales
- Total Profit
- Total Orders
- Total Quantity

## Filters
- Region
- Category

## Visualizations
- Pie Chart: Sales Distribution by Category
- Column Chart: Sales by Category
- Column Chart: Profit by Region

## Tools Used
- Power BI
- Excel Dataset
